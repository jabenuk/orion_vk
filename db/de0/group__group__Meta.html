Detailed Description

Functionality found in this module is related to initialising, configuring, managing, terminating, etc.

It is part of the core Orion library interface.

Typedef Documentation

◆ oriState

typedef struct oriState oriState

This structure is mostly used to store lists of created Orion objects so they can be implicitly destroyed in oriTerminate().

However, it also holds state related to Vulkan, such as the application info.

Note that Vulkan objects, like instances, are not stored inside oriState, but rather this struct contains handles to said Vulkan objects, so that it can manage them. For example, you can create a Vulkan instance using data from an oriState object with oriCreateInstance(), but the function returns the instance so you can deal with it on your end. However, the function also returns the instance to the state, so that, when you call oriDestroyState(), the instance (and any other instances made with this state) will implicitly be freed.

◆ oriDestroyState()

void oriDestroyState ( oriState *  state)

Calling this function clears all data stored in state and destroys all Orion and Vulkan objects registered to it.

Warning
If you destroy a Vulkan object that was created with state before calling this function, make sure to set the handle to the object to NULL (e.g. destroying a logical device -> set the VkDevice struct to NULL). Otherwise, an exception will occur when this function is called.
Parameters
statehandle to the state object to be destroyed.

◆ oriSetVulkanAllocationCallbacks()

void oriSetVulkanAllocationCallbacks ( VkAllocationCallbacks  callbacks)

This function sets the internally-held structure in which allocation function pointers can be defined.

The structure passed to the callbacks parameter of this function will be referenced in any Vulkan function called internally by the library with a pAllocator parameter.

Passing NULL to this function will reset Vulkan to using default allocation callbacks as described by the implementation.

Parameters
callbacksa pointer to the callbacks structure to use with Vulkan functions.
