void oriDefineStateInstanceEnabledDebugMessages ( oriState *  state,
VkDebugUtilsMessageSeverityFlagBitsEXT  severities,
VkDebugUtilsMessageTypeFlagBitsEXT  types 
)

This function only has an effect if the VK_EXT_debug_utils extension has been specified. This must be done before the respective call to oriCreateStateInstance().

If you call this function, the ORION_FLAG_CREATE_INSTANCE_DEBUG_MESSENGERS flag will be implicitly set to true.

Note
The message filters specified in this function are different to those specified in oriEnableDebugMessages(). Whilst those specified in that function are related to the Orion library, those specified here are related to the Vulkan API.
Parameters
statethe state to modify.
severities(bitmask) severities of the messages to display. By default, no messages are displayed (even if ORION_FLAG_CREATE_INSTANCE_DEBUG_MESSENGERS is true)
types(bitmask) types of the messages to display. By default, no messages are displayed (even if ORION_FLAG_CREATE_INSTANCE_DEBUG_MESSENGERS is true)
