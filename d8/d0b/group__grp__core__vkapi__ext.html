Detailed Description

Functionality found in this module is related to layers and extensions in Vulkan. It is part of the core Vulkan abstractions.

One guide to summarise the use of layers can be found in the Vulkan-Guide GitHub repo. This answer on StackExchange suitably describes the use of Vulkan extensions.

See also
KhronosGroup/Vulkan-Guide

◆ oriCheckInstanceExtensionAvailability()

const bool oriCheckInstanceExtensionAvailability ( const char *  extension,
const char *  layer 
)

This function queries if the given instance extension is provided by either the Vulkan implementation or the given layer, returns true if it is, and returns false if otherwise.

If layer is NULL, Orion will check if the extension is provided by your machine's Vulkan implementation. Otherwise, Orion will check if the extension is provided by layer.

False will be returned in the event of an error, or if layer is invalid.

Parameters
extensionthe name of the instance extension to check.
layerNULL or the name of the providing layer.
Returns
true if the extension is provided.
false if the extension is not provided, or if there was an error.
